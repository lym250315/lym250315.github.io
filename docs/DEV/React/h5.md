## HTML basic

### head

```html
<!DOCTYPE html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <!-- icon 的 href 给一个 64x64 的 png 就可以了，会在浏览器的 tab 显示 -->
    <link rel="icon" href="..." />
    <!-- title 会在浏览器的 tab 中显示 -->
    <title>...</title>
    <meta name="author" content="..." />
    <!-- description 会在搜索引擎的搜索首页出现在标题下方 -->
    <meta name="description" content="..." />
    <!-- 不需要 keywords，因为搜索引擎不看 -->
  </head>
</html>
```
